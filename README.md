# 🧬 BioLab Calculator

A comprehensive **Python + Streamlit** application that provides essential laboratory calculators for **biotechnology students, researchers, microbiologists, molecular biologists, and laboratory professionals**.

Designed to simplify routine laboratory calculations, BioLab Calculator combines multiple scientific tools into one intuitive and interactive application.

---

## 📌 Features

### 🧪 Solution Preparation
- Molarity Calculator
- Normality Calculator
- PBS Preparation Calculator
- Agar Preparation Calculator
- Media Preparation Calculator

### 💧 Dilution
- Dilution Calculator (C₁V₁ = C₂V₂)

### 🧬 Molecular Biology
- PCR Master Mix Calculator
- DNA Concentration Calculator
- RNA Concentration Calculator
- DNA Dilution Calculator
- Serial Dilution Calculator

### 📏 Laboratory Utilities
- Unit Converter
- Molecular Weight Calculator

### ℹ️ Additional
- Professional Home Dashboard
- About Page
- Responsive Streamlit Interface

## 🚀 Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/harminderkaur1329/BioLab Calculator.git
```

### 2️⃣ Navigate to the project

```bash
cd BioLab Calculator
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the application

```bash
streamlit run app.py
```

## 💡 Why This Project?

Laboratory calculations are frequently repeated during experiments and are prone to manual calculation errors.

BioLab Calculator was developed to provide a simple, accurate, and user-friendly platform for performing common laboratory calculations efficiently.
